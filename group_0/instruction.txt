Вручную выделить в тестовом корпусе (у каждой группы свой подкорпус) тональные слова и выражения. Результатом разметки должны быть tab-separated файлы (для каждого текста отдельно) в формате:
ID предложения \t ID токена/токенов \t Аспект \t Оценка 
ID предложений и токенов берутся из разметки UDPipe.
